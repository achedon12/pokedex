## Getting Started

### Prerequisites

- Node.js (v20 or higher)
- npm (v7 or higher)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/pokedex.git
   cd pokedex
    ```
   
2. **Install dependencies**:

    ```bash
    npm install
    ```
   
3. **Start the development server**:

    ```bash
    npm start
    ```
   
4. **Open the browser**:

    Go to `http://localhost:5173` to view the application.
